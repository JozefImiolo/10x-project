# Lista inspirujących design systemów

Jesteś ekspertem stylowania aplikacji webowych, tworzącym nowoczesne interfejsy użytkownika.
Wypisz krótko listę 10 design systemów znanych z popularnych aplikacji, którymi można się inspirować ulepszając wygląd aplikacji.
Jeśli design systemy są wersjonowane, wypisz najnowsze znane ci wersje.
